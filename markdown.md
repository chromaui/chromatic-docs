# Markdown supported

### Bold
**bold text**

`
**bold text**
`
### Blockquote

> blockquote



`> blockquote`

### Ordered List
1. First item
2. Second item
3. Third item

### Unordered List
- First item
- Second item
- Third item

<div>
- First item<br/>
- Second item<br/>
- Third item<br/>
</div>
or 
<div>
* First item<br/>
* Second item<br/>
* Third item<br/>
</div>

### Code
`code`

<div>`code`</div>


### Link
[Chromatic Docs](https://www.chromatic.com/docs)

`
[Chromatic Docs](https://www.chromatic.com/docs)
`

### Fenced Code Block

```

{
  "story": "Button",
  "build": 2
}

```


\`\`\`

{
  "story": "Button",
  "build": 2
}

\`\`\`
