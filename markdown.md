# Markdown supported

### Bold
**bold text**

```
**bold text**
```
### Blockquote

> blockquote
```
> blockquote
```
### Ordered List
1. First item
2. Second item
3. Third item
```
1. First item
2. Second item
3. Third item
```
### Unordered List
- First item
- Second item
- Third item

```
- First item
- Second item
- Third item
```
### Code
`code`

```
`code`
```


### Link
[Markdown Guide](https://www.markdownguide.org)

```
[Markdown Guide](https://www.markdownguide.org)
```

### Fenced Code Block

```
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
