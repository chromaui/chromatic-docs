# How do I set up notifications?

To set up notifications on your account, you'll need to do the following:

1. Open the Notifications page: https://www.chromatic.com/notifications

2. Indicate the email address where you want to receive the notifications for each project. **Please note that emails can be designated per project.** 

3. Set the notifications you want to set for build status, reviews, and discussions.
