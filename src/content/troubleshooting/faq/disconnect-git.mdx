# Why can't I disconnect my Git user from my Chromatic profile?

When you create your Chromatic profile using a git provider, your Git user details are used to log in, so disconnecting the Git provider won't be possible unless you link another Git provider.
To prevent that, create a new email/password user and link it to your Git provider.

**Note:** User profiles cannot be converted from Git provider to email/password.
