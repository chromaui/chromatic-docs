# Do I get charged for snapshots when I rerun a build?

When you rerun a build by clicking the rerun button on the UI Test page, Chromatic only bills snapshots taken for denied and unreviewed tests. Snapshots for accepted tests and TurboSnaps are not retaken, so they are not billed again.
