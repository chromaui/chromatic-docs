# How do we run an interaction test when snapshots are disabled for a story?

The `disableSnapshot` parameter will prevent Chromatic from capturing the snapshot. When you disable the snapshots, the Interaction tests are automatically disabled. The interaction indicator on your builds page will only be relevant to the snapshotted stories.
