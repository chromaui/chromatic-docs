# Parameter types

| Parameter                                                                                                | Type       | Explanation                         |
| -------------------------------------------------------------------------------------------------------- | ---------- | ----------------------------------- |
| [`delay`](/docs/delay/)                                                                                  | `number`   | Wait time before capture            |
| [`a11y`](/docs/accessibility/configure/#disable-accessibility-tests)                                     | `boolean`  | Disable accessibility tests         |
| [`diffIncludeAntiAliasing`](/docs/threshold#anti-aliasing)                                               | `boolean`  | Include anti-aliasing in diffs      |
| [`disableSnapshot`](/docs/disable-snapshots)                                                             | `boolean`  | Skip visual tests                   |
| [`diffThreshold`](/docs/threshold#setting-the-threshold)                                                 | `number`   | Visual change sensitivity           |
| [`viewport`](/docs/modes/viewports/)                                                                     | `object`   | Default viewport settings           |
| [`ignoreSelectors`](/docs/ignoring-elements)                                                             | `string[]` | Ignore elements                     |
| [`forcedColors`](/docs/media-features#test-high-contrast-color-schemes)                                  | `string`   | Forced colors mode                  |
| [`pauseAnimationAtEnd`](/docs/animations#css-animations)                                                 | `boolean`  | Pause at animation end              |
| [`prefersReducedMotion`](/docs/media-features#verify-reduced-motion-animations)                          | `string`   | Reduce motion setting               |
| [`media`](/docs/media-features/#combine-media-features-with-modes)                                       | `string`   | Simulates print query               |
| [`cropToViewport`](docs/modes/viewports/#how-does-snapshot-cropping-work-with-viewport-width-and-height) | `boolean`  | Crop the screenshot to the viewport |
